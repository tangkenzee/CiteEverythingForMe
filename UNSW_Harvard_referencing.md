# UNSW Harvard Referencing Guide - Websites

This guide follows the UNSW (University of New South Wales) Harvard referencing style for websites.

## In-Text Citations

Cite the name of the author/organisation responsible for the site and the date created or last revised (use the most recent date):

**Format:**
```
(Author/Organisation Year)
```

**Example:**
```
(Department of Social Services 2020)
```

**Alternative format (narrative citation):**
```
According to the Department of Social Services (2020) ...
```

---

## List of References

Include information in the following order:

1. **Author** - The person or organisation responsible for the site
2. **Year** - Date created or revised
3. **Site name** - In italics
4. **Name of sponsor of site** - If available
5. **Accessed day month year** - The date you viewed the site
6. **URL or Internet address** - Between pointed brackets. If possible, ensure that the URL is included without a line-break.

### Format

```
Author Year, Site name (in italics), Name of sponsor (if available), accessed Day Month Year, <URL>.
```

### Example

```
Department of Social Services 2020, Department of social services website, Australian government, accessed 20 February 2020, <https://www.dss.gov.au/>.
```

---

## Notes

- Use the most recent date available (creation date or last revised date)
- Always include the access date (the date you viewed the site)
- URLs should be enclosed in pointed brackets: `<https://example.com>`
- Try to keep URLs on a single line without breaks
- If no author/organisation is available, use the site name as the author
